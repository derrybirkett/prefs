# Session Summary

## March 20, 2026

### Bloom System

**Components:**
- **seed** - CLI that bootstraps projects from intent
- **prefs** - Philosophy and preferences
- **hatch** - Stack scaffold
- **grove** - Agentic layer (replaces pip)

**Architecture:**
- prefs is a submodule
- Users can have their own prefs repo
- seed is agnostic - reads prefs, calls grove and hatch
- Hatch is opinionated but prefs wins when specified

### Key Commands

```bash
# Full Bloom (default)
seed init --story "..."

# Without LLM parsing
seed init --no-llm --story "..."

# Skip components
seed init --story "..." --no-grove
seed init --story "..." --no-prefs
```

### Preferences

- Tech: NX monorepo, Playwright, shadcn, Docker
- Build: CLI → API → UI
- Design: Minimalism, black/white, error red, success green
- Workflow: Feature branches, squash commits, wrap up command
