# User Preferences for Digital Product Development

This repository contains my preferences, methodologies, and conventions for building digital products. It serves as a reference for AI agents and development tools to understand how I like to work.

## Structure

- `CLAUDE.md` - Instructions for Claude Code and AI agents
- `methodology.md` - Product development philosophy and approaches
- `tech-stack.md` - Technology choices and architecture preferences
- `workflows/` - Git workflows, automation, and development processes
- `product-surfaces/` - Standard product components and patterns

## Quick Reference

**Philosophy**: Learning organizations, lean methodology, design thinking
**Stack**: NX monorepos, Playwright, shadcn/ui
**Workflow**: Branch-based development with GitHub issues and PRs
**Core Flow**: Landing page → Login → Dashboard → Logout

See individual files for detailed preferences.
